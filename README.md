# machine-learning-toolset

This repository is for creating the toolset for machine learning colearning

# Setup environment

``` bash
sudo apt update
sudo apt install -y python3-dev python3-pip
sudo pip3 install -U virtualenv  # system-wide install

python3 --version
pip3 --version
virtualenv --version


```
